Notion-Style Landing Page (Canvas UI)
This project showcases a modern, Notion-inspired landing page designed entirely using the HTML5 Canvas API.
The layout, typography, and UI elements are rendered programmatically without relying on traditional HTML components or external libraries.

Overview

The goal of this project is to demonstrate how a complete marketing-style layout can be created using a single canvas element. It focuses on clean design, spacing, and visual hierarchy while keeping the implementation lightweight and self-contained.

Features

Canvas-rendered landing page layout

Gradient background with subtle shadow effects

Header navigation with call-to-action button

Hero section with clear messaging

Primary and secondary action buttons

Simple illustration drawn using canvas primitives

Single-file implementation

Technologies Used

HTML5

CSS3

JavaScript (Canvas 2D API)

Project Structure
index.html

All styling and rendering logic is contained within the same file for ease of use and portability.

Getting Started

Clone or download the repository.

Open index.html in any modern web browser.

The canvas content renders automatically on load.

No additional setup or dependencies are required.

Use Cases

Learning and practicing the HTML5 Canvas API

Prototyping UI layouts visually

Understanding canvas-based rendering workflows

Design experimentation without frameworks

Limitations

Fixed canvas dimensions

No interactivity or responsiveness

Text and UI elements are not selectable

Possible Enhancements

Responsive canvas scaling

Interactive buttons and hover states

Animations and transitions

Migration to semantic HTML and CSS
